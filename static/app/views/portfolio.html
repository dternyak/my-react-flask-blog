GetDotBit

GetDotBot's features include secure user registration, a custom bitcoin powered payment system, linux shell scripting to namecoind process, and a great front-end powered with Bootstrap and built using Ajax to reduce client reloading. It makes use of a wide array of API’s including ShapeShift.io, Blocktrail.com, Blockchain.info and CryptoCoinCharts.com to build a robust yet user-friendly site. It uses best practices like form validation and password hashing to provide a professional experience. Included in the stack are: Ubuntu, Nginx, Flask, SQLite, JQuery, Bootstrap.
