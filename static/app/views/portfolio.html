MoneroWallet

MoneroWallet is an AngularJS based CryptoCurrency Web Wallet for storing and sending Monero. It uses Firebase for its backend, and Google App Engine for hosting. Firebase works by storing an encrypted seed generated by the client. After authentication, it is retrieved and is used to rebuild the Monero Private Key. This is used to sign transactions which are later broadcast to the Monero Network. An Android app was also built leveraging the same back-end.

BitZipped

BitZipped is a RESTfully designed web application for selling digital content using Bitcoin. Users can register and upload their content, customize the price, and set a banner image they would like to display with their content. Customers can navigate to a unique link for each product, and upon payment of the described price, will receive a unique URL to access the content that expires after the download.

CryptoIndex

CryptoIndex is a dashboard for viewing CryptoCurrency prices. It uses AngularJS and is a completely static site, leveraging external API's to provide price data. It is similarly hosted on Google App Engine, and optimized for static loading.
